# NSF HERD AI Assistant

Query 15 years of university R&D data using natural language.

## Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features
- Natural language queries
- Auto-generated visualizations
- CSV export
- 1,004 institutions, 2010-2024

## Example Questions
- What is UNT's total R&D for 2024?
- Compare UNT and Texas Tech
- Show top 5 Texas universities

## Data Source
NSF HERD Survey (2010-2024)
